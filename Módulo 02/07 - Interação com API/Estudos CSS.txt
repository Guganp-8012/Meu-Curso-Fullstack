Background (plano de fundo do elemento):

background-color - Define a cor de fundo de um elemento

background-image - Define a imagem de fundo de um elemento
A imagem de fundo também pode ser definida para elementos específicos como <p>

background-repeat - Define como uma imagem de fundo será repetida
Por padrão a imagem de fundo já é repetida tanto na horizontal quanto na vertical
A imagem de fundo só pode ser repetida na vertical e na horizontal
no-repeat - Define para que a imagem seja repetida uma única vez

background-position - Define a posição inicial de uma imagem de fundo
ex.: background-position: right top;

background-attachment - Define se uma imagem de fundo é fixa ou acompanha o resto da página conforme seja rolada
fixed/scroll

background (propriedade abreviada) - Define todas as propriedades de background em uma declaração
ex.: background: #ffffff url("img_tree.png") no-repeat right top;


Borders (bordas):
As propriedades da borda permitem especificar o estilo, a largura e a cor da borda de um elemento.

border-style - Específica qual tipo de borda vai ser exibida
dotted - pontilhada
dashed - tracejada
solid - sólido
double - dobrada
groove - ranhurada/entalhada (um tipo de borda 3D, o efeito depende do valor da cor da borda)
ridge - estriada (um tipo de borda 3D, o efeito depende do valor da cor da borda)
inset - inserida (um tipo de borda 3D, o efeito depende do valor da cor da borda)
outset - inicial (um tipo de borda 3D, o efeito depende do valor da cor da borda)
none - sem bordas/nenhuma
hidden - oculta/escondida

Também é possível definir mais de um tipo de borda ao mesmo tempo, "misturando-as"
ex.: border-style: dotted dashed solid double

border-width - Específica a largura/grossura das bordas
*também pode definir um valor personalidado para cada borda se preferir

border-color - Defini a cor das quatro bordas
*também pode definir um valor personalidado para cada borda se preferir

border - Lados Individuais
Em CSS, também existem propriedades para especificar cada uma das bordas (superior, direita, inferior e esquerda)
ex.:

p {
  border-top-style: dotted;
  border-right-style: solid;
  border-bottom-style: dotted;
  border-left-style: solid;
}


Border (propriedade abreviada)
A propriedade border é uma propriedade abreviada para as seguintes propriedades de borda individuais:
largura da borda; estilo de borda (obrigatório); Cor da borda.

border-radius - é usado para adicionar bordas arredondadas a um elemento

Margin (Margem):
As margens são usadas para criar espaço ao redor dos elementos, fora de quaisquer bordas definidas.

margin - Lados Individuais:

margin-top; margin-right; margin-bottom; margin-left

---> (não sei como ajeitar essa parte)

Todas as propriedades de margem podem ter os seguintes valores:

    auto - o navegador calcula a margem
    comprimento - especifica uma margem em px, pt, cm, etc.
    % - especifica uma margem em% da largura do elemento que contém
    herdar - especifica que a margem deve ser herdada do elemento pai
Dica: Valores negativos são permitidos. <---
